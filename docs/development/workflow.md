# Development Workflow

## Setup

```bash
bun install
``

## Validate

```bash
bun run check
``

## Format

```bash
bun run format
``

## Typecheck

```bash
bun run typecheck
``

## Commit

Use Conventional Commits.

```bash
git add .
git commit -m "type(scope): summary"
``

## Rule

Add an ADR before introducing major tools, architectural structure, or workflow conventions.
