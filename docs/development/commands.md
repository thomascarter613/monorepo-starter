# Commands

## Install

```bash
bun install
``

## Validation

```bash
bun run check
``

## Formatting

```bash
bun run format
bun run format:check
``

## Linting

```bash
bun run lint
``

## Typechecking

```bash
bun run typecheck
``

## Testing

```bash
bun run test
``

## Build

```bash
bun run build
``

## Moon

```bash
moon run root:check
moon run root:typecheck
``
